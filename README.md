## Features
- **Responsive Sidebar**: Fixed sidebar on desktop, collapses to hamburger menu on mobile
- **Dark/Light Mode Toggle**: Smooth theme switching with persistent states
- **5 Color Themes**: Style switcher with different color schemes (Red, Blue, Orange, Green, Pink)
- **Typing Animation**: Dynamic text animation on the home section
- **Smooth Navigation**: Click links to scroll between sections
- **Skills Progress Bars**: Visual representation of technical skills
- **Services & Projects Sections**: Dedicated areas to showcase my work
- **Contact Form**: Functional form UI (backend integration coming soon)
- **Ripple Effect**: Visual feedback when clicking navigation items

## What I Used
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations, Media Queries)
- JavaScript (Vanilla)
- Typed.js library for typing animation
- Font Awesome icons

## Things I Learned / Practiced

**JavaScript**
- DOM manipulation for sidebar toggling
- Managing active states for navigation links
- Typed.js integration for text animation
- Style switcher functionality (theme colors)
- Dark mode implementation with class toggling
- Ripple effect on click events
- Smooth scroll behavior
- Event delegation for navigation

**CSS**
- CSS Variables for theming (light/dark mode + 5 color skins)
- Media queries for responsive breakpoints
- Flexbox and Grid layouts
- Keyframe animations (typing, ripple, transitions)
- Pseudo-elements (`::before`, `::after`) for decorative borders
- Z-index management for layered UI
- Custom scrollbar styling
- Backdrop filter for blur effects
- Gradient backgrounds and text effects

**Advanced Concepts**
- Multiple theme management (dark/light + color skins)
- Mobile-first responsive approach
- Position fixed for sidebar
- CSS transitions for smooth interactions
- Modular JS files (separated by functionality)

## How to Run
1. Clone the repo
2. Open `index.html` in any browser
3. Click the gear icon (top-right) to access theme switcher
4. Try different colors and dark/light modes
5. On mobile, use hamburger icon to open sidebar
